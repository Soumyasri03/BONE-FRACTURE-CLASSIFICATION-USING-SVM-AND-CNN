🦴 Bone Fracture Detection & Classification (CNN + SVM)

- This project is an AI-based system that detects and classifies bone fractures from X-ray images. It uses a CNN for        feature  extraction and an SVM classifier for final prediction, providing faster and more accurate fracture identification.

⭐ Features

- Upload an X-ray image to get fracture classification

- CNN model extracts visual features from X-ray images

- SVM improves classification accuracy over CNN alone

- Includes preprocessing, evaluation, and comparison of models

- Simple Python-based interface for predictions

🛠 Technologies

- Python

- TensorFlow/Keras

- Scikit-learn

- OpenCV

- NumPy, Pandas

- Matplotlib

🔄 Workflow

- Preprocess X-ray images

- Train CNN for feature extraction

- Extract deep features from CNN

- Train SVM classifier with extracted features

- Compare CNN vs. CNN+SVM

- Predict fracture type from uploaded images

📊 Evaluation

- Accuracy

- Precision

- Recall

- F1-score
